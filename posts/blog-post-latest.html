A Front end Journey

March 31, 2023 | Premkumar Shanmugam
Share this blog post:
| | | |

I am super excited to embark on this front end journey, leveraging the Bertelsmann's Next Gen Tech Booster Scholarship on Udacity's Front End Developer program.

This is my third nano degree program on Udacity, and second Bertelsmann scholarship. I have completed the nano degrees Intro to Machine Learning using PyTorch and Azure Architect using Microsoft Azure in previous years.

As with all the scholarship programs, this one too started with a challenge course carved out from the nano degree program. Having used javascript frameworks at work, it was interesting to understand how the core underlying tools - HTML, CSS and vanilla JavaScript (more of this coming in the days ahead) - dance together to help the frameworks render the magic on browser UIs. I could not help but allow Daniel's (HTML, CSS content instructor) super-contagious enthusiasm rub off on me. Thank you, Daniel 🙏.

I went through all the class videos once again to refresh on HTML and CSS. I tried to experiment with most of the learnings from the first module. While I was at it, I also learned about many more flexbox features from css-tricks.com and tried flex-flow in this project. With this I'm submitting the project for review and keeping 🤞.

Wish me luck!
